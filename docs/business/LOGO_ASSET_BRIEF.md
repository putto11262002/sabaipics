## Brand Context

### What is SabaiPICS?

SabaiPICS (สบายพิกส์) is an event photo distribution platform for Thai photographers.

- **"Sabai"** = Easy, relaxed, comfortable (สบาย)
- **"PICS"** = Photos

**Core promise:** Make photo delivery sabai (effortless) for photographers and participants.

### What We Do
1. Photographer uploads event photos
2. AI indexes faces automatically
3. Participants find their photos via LINE (selfie search)
4. Photos delivered instantly

### Brand Personality
- Approachable, not salesy
- Straightforward, no jargon
- Confident but humble
- Photographer-focused

### Key Differentiators
- Transparent pricing (no hidden fees)
- All features included (no tier gating)
- Reliable infrastructure
- Lowest entry cost in market (฿50 vs ฿450+ competitors)

## Logo Requirements

### Logo Concept Direction

**Name breakdown:**
- "Sabai" - Relaxed, easy, comfortable feeling
- "PICS" - Short for pictures, modern/casual

**Possible approaches:**
1. **Wordmark only** - Clean typography, "Sabai" + "PICS" as one word or separated
2. **Lettermark + Wordmark** - "S" or "SP" icon with full name
3. **Symbol + Wordmark** - Abstract mark representing ease/photos + wordmark

**Symbol ideas (if using icon):**
- Simplified camera aperture (photography)
- Face silhouette with frame (face recognition)
- Flowing/easy shape (sabai feeling)
- Photo frame with checkmark (delivery complete)
- Abstract "S" that suggests flow/ease

### Logo Style

| Attribute | Direction |
|-----------|-----------|
| Style | Minimal, modern, geometric |
| Feel | Professional yet friendly |
| Complexity | Simple enough to work at 16px |
| Weight | Medium - not too thin, not too heavy |

**Reference brands:** Notion, Linear, Vercel, Stripe

### Logo Versions Required

| Version | Use Case |
|---------|----------|
| **Primary (Horizontal)** | Website header, documents |
| **Stacked (Vertical)** | Square spaces, app stores |
| **Icon only** | Favicon, app icon, avatars |
| **Wordmark only** | When icon is already visible |

### Color Versions

Each logo version needs these color variants:

| Variant | Colors | Use Case |
|---------|--------|----------|
| Full color on light | Black + Accent (#9B8FC7) | White/light backgrounds |
| Full color on dark | White + Accent (#9B8FC7) | Dark backgrounds |
| Monochrome black | #1A1A1A only | Documents, single-color |
| Monochrome white | #FFFFFF only | Dark backgrounds, overlays |

## Design Do's and Don'ts

### DO
- Use white space generously
- Keep the logo clean and minimal
- Ensure icon works at 16px (favicon size)
- Test on both light and dark backgrounds
- Make "PICS" feel connected to "Sabai" (one brand, not two words)
- Consider Thai market context (LINE-heavy, mobile-first)

### DON'T
- Use gradients in the logo
- Make the icon too detailed/complex
- Use more than 2 colors in the logo
- Add taglines to the logo lockup
- Use trendy effects (glows, 3D, etc.)
- Make "Sabai" and "PICS" feel disconnected
