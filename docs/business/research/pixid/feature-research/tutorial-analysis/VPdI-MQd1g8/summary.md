## Video Summary

### Features Mentioned
| Feature | What They Say About It |
|---------|------------------------|
| FTP Upload from Camera | Direct photo upload from Nikon cameras (Z8, Z9) without USB cable connection |
| Wi-Fi Connectivity | Camera can connect to Wi-Fi networks for wireless upload |
| Ethernet Connectivity | Alternative wired network option for FTP upload |
| FTP Server Integration | Pixid provides FTP server credentials for photographers to configure in-camera |
| Automatic Upload | Camera can be configured to automatically send photos after capture |
| File Format Selection | Supports uploading JPEG, RAW, or both formats |
| File Size/Quality Control | Photographers can configure image size and quality for upload |
| Destination Folder Configuration | Ability to set specific folder paths on FTP server for organizing uploaded files |
| Connection Testing | Built-in test connection feature to verify FTP setup |
| Profile Management | Create and save multiple FTP connection profiles in camera |

### Workflows Demonstrated
| Workflow | Steps Described |
|----------|-----------------|
| Initial FTP Setup in Nikon Camera | 1. Check camera model compatibility (Z8, Z9)<br>2. Navigate to Set up menu > Network > FTP upload<br>3. Create new profile (e.g., "Pixid_Upload")<br>4. Configure network settings (Wi-Fi or Ethernet)<br>5. Enter Wi-Fi credentials if using wireless<br>6. Input FTP server settings (address, username, password)<br>7. Set destination folder on server<br>8. Select file format (JPEG/RAW/both)<br>9. Configure file size and quality<br>10. Save settings and test connection<br>11. Enable automatic upload after capture |
| Getting FTP Credentials | Contact Pixid to obtain FTP server details (server address, username, password) |

### Marketing Claims / Pain Points Addressed
- **Cable-free workflow**: "No need to use USB cable" - eliminates physical connection requirement
- **Speed**: "Makes photo sending faster" when using FTP with Pixid
- **Convenience**: "More convenient" upload process compared to traditional methods
- **Direct upload**: Photographers can send photos directly from camera to platform
- **Immediate availability**: Photos can be uploaded automatically right after capture, reducing time-to-distribution
