## Video Summary

### Features Mentioned
| Feature | What They Say About It |
|---------|------------------------|
| Desktop Uploader Application | Same version as web app, can log in with username/password. Has device locking system - when logging in on new device, old device gets logged out automatically |
| Upload Quality Settings | Can choose upload size: 1,500, 2,000, or 4,000 pixels. Recommends 2,000 pixels for balance between quality and mobile loading speed |
| Lightroom Integration | Fixed bug for Lightroom connection. Supports auto-upload from Lightroom. Works with tethered shooting |
| Folder Monitoring | Can select desktop folder for monitoring images. Pixid monitors only designated folders. Auto-uploads from monitored folders |
| Camera Tethering | Recommends specific hardware (Tester 2 cable/orange cable) for connection. Has auto-upload function with Lightroom. Real-time access to photos in gallery |
| QR Code Gallery System | Can create QR codes for galleries. Customization options (fonts, colors, headers). Gallery management features |
| Personal Photo QR Codes | Individual QR codes for selected photos |
| Mobile Download Experience | Android: Direct download with orange button. iPhone: Long press then save image |

### Workflows Demonstrated
| Workflow | Steps Described |
|----------|-----------------|
| Initial Setup | 1. Install desktop uploader application 2. Log in with username/password (same as web) 3. Set upload quality (recommended 2,000px) |
| Folder Monitoring Setup | 1. Select folder on desktop for monitoring 2. Pixid monitors designated folders only 3. Auto-upload activates for monitored folders |
| Tethered Shooting Workflow | 1. Connect camera with recommended cable (Tester 2/orange cable) 2. Connect to Lightroom 3. Enable auto-upload 4. Photos appear in gallery in real-time |
| Gallery Creation & Sharing | 1. Create gallery through web interface 2. Generate QR code for gallery 3. Customize QR code (fonts, colors, headers) 4. Share QR code with attendees |
| Attendee Photo Retrieval | 1. Scan QR code to access gallery 2. Browse photos 3. Select desired photos 4. Android: Direct download via orange button / iPhone: Long press and save |

### Marketing Claims / Pain Points Addressed
- **Seamless real-time delivery**: Photos accessible in gallery immediately during tethered shooting
- **Quality vs. speed optimization**: 2,000px recommendation balances image quality with mobile loading performance
- **Device security**: Automatic device locking prevents unauthorized concurrent access
- **Professional workflow integration**: Native Lightroom support for photographers already using industry-standard tools
- **Simple attendee experience**: QR code scanning provides instant gallery access without app installation
- **Cross-platform compatibility**: Works on both Android and iPhone with platform-specific optimizations
- **Customizable branding**: QR code customization allows photographers to maintain brand consistency
