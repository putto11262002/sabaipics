## Video Summary

### Features Mentioned
| Feature | What They Say About It |
|---------|------------------------|
| Sony Image Edge Desktop Integration | Required software to connect Sony cameras to computer via USB for remote shooting and automatic file transfer |
| Lightroom Plugin (Pixid) | Plugin installed via File > Plug-in Manager that enables Pixid integration within Lightroom workflow |
| Tethered Capture | Lightroom native feature used to automatically import photos as they're shot, configured to watch the same folder as Sony Image Edge Desktop |
| Remote Shooting | Control camera from computer via Sony Image Edge Desktop software |
| Dual File Storage | Simultaneously saves photos to computer and camera memory card |
| Automatic Upload | Pixid plugin automatically uploads photos to their system directly from Lightroom |

### Workflows Demonstrated
| Workflow | Steps Described |
|----------|-----------------|
| Initial Setup | 1. Download Sony Image Edge Desktop from Sony website (select correct camera model)<br>2. Install the software<br>3. Connect Sony camera to computer via USB cable<br>4. Install Pixid Lightroom plugin via File > Plug-in Manager > Add<br>5. Select .lrplugin file<br>6. Restart Lightroom |
| Shooting Configuration | 1. Open Sony Image Edge Desktop<br>2. Select Remote/Remote Shooting mode<br>3. Configure to save as JPEG files<br>4. Set dual storage: computer + memory card<br>5. Create dedicated folder for photo storage<br>6. Configure Sony software to save to this folder |
| Lightroom Setup | 1. Go to File > Tethered Capture > Start Tethered Capture<br>2. Configure to save files to same folder as Sony Image Edge Desktop<br>3. Set filename and formatting preferences |
| Live Shooting Workflow | 1. Take photo with Sony camera<br>2. Photo automatically transfers to computer via Sony Image Edge Desktop<br>3. Photo simultaneously appears in Lightroom via Tethered Capture<br>4. Use Pixid plugin to automatically upload to their system |

### Marketing Claims / Pain Points Addressed
- Solves compatibility issue between Sony cameras and Lightroom's native tethered capture (which doesn't support Sony directly)
- Enables professional photographer workflow: shoot → auto-import → auto-upload to distribution platform
- Streamlines event photography workflow by eliminating manual file transfers
- Provides redundancy with dual storage (computer + memory card)
- Integrates photo distribution directly into existing Lightroom editing workflow
