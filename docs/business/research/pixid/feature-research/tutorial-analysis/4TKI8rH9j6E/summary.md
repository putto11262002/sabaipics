## Video Summary

### Features Mentioned
| Feature | What They Say About It |
|---------|------------------------|
| Transfer & Tagging Mobile App | Mobile app for transferring photos from Sony cameras via Wi-Fi; available on App Store and Google Play |
| Wi-Fi Camera Connection | Connect Sony camera to smartphone via Wi-Fi without cables; uses QR code scanning for easy pairing |
| Selective Photo Transfer | Photographers can select which images to send from camera to phone/server |
| File Format Options | Supports JPEG transfer and custom size settings for optimized file delivery |
| Transfer Modes | Multiple transfer mode options available in the app for different use cases |
| Real-time Photo Transfer | Supports shooting and sending photos in real-time over Wi-Fi connection |
| Automatic Upload | Can be configured to automatically send photos after capture |
| File Management | Built-in file management and configuration settings |
| Metadata Enrichment | Ability to add GPS location, keywords, and other metadata to images during transfer |
| FTP Integration | T&T FTP functionality for direct server uploads |
| Folder Routing | Can configure app to send photos to specific folders (important for Pixid integration) |

### Workflows Demonstrated
| Workflow | Steps Described |
|----------|-----------------|
| Initial Setup | 1. Download Transfer & Tagging Mobile app from store<br>2. Enable Wi-Fi on Sony camera<br>3. Connect camera to smartphone via Wi-Fi<br>4. Navigate to Menu > Network > Send to Smartphone > Select on Smartphone<br>5. Select camera in T&T app<br>6. Scan QR code on camera screen to pair |
| Photo Transfer to Pixid | 1. Connect camera via T&T app<br>2. Configure app to send to designated folder (Pixid integration)<br>3. Set file format preferences (JPEG, size)<br>4. Select transfer mode<br>5. Select photos to transfer or enable auto-transfer |
| Real-time Event Shooting | 1. Establish Wi-Fi connection between camera and phone<br>2. Enable automatic upload after capture<br>3. Shoot photos that automatically transfer in real-time<br>4. Photos route to configured destination (e.g., Pixid workflow) |

### Marketing Claims / Pain Points Addressed
- Wireless workflow eliminates need for cables
- Fast transfer speeds via Wi-Fi
- Real-time delivery enables immediate photo availability at events
- Seamless integration with Pixid platform for event photography workflow
- Comprehensive file management and configuration options
- Works with multiple Sony camera models that have built-in Wi-Fi
- Reduces friction in photographer workflow from capture to distribution
