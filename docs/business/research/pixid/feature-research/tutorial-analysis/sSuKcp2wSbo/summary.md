## Video Summary

### Features Mentioned
| Feature | What They Say About It |
|---------|------------------------|
| Logo Upload | Supports PNG or JPEG formats with transparent backgrounds. Can upload brand logo/symbol in Branding section |
| Logo Positioning & Sizing | Can configure position and size of logo as desired |
| Logo Opacity | Can adjust logo transparency/opacity as appropriate |
| Brand Text Overlay | Can add text or brand name to gallery |
| Text Styling | Can configure color and font of text to align with brand identity |
| Text Positioning | Can choose position for text display - top, bottom, or sides of image |
| Cover Frame Templates | Can create branded cover frame templates |
| Custom Frame Upload | Can upload Pixid-provided template frames or create custom ones |
| Frame Width & Logo Placement | Can configure frame width and logo position within frame |
| Contact Information Display | Can add brand contact info like phone number or website |
| Contact Info in Gallery | Can configure contact information to display in gallery |
| Branding Presets | Can save branding settings as presets for reuse |
| Preview Gallery | Can test display by creating sample gallery |
| Brand Preview & Adjustment | Can preview branded images and adjust as needed |
