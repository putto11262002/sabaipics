# ยินดีต้อนรับสู่ FrameFast Docs

เรียนรู้วิธีใช้ FrameFast เพื่อแจกจ่ายรูปภาพงานอีเวนต์ด้วยระบบจดจำใบหน้า AI

## เริ่มต้นใช้งาน

เริ่มใช้งาน FrameFast ได้ภายในไม่กี่นาที

- **อัปโหลดรูปภาพ** - เรียนรู้วิธีต่างๆ ในการอัปโหลดรูปภาพงานอีเวนต์
- **แชร์กับผู้เข้าร่วม** - ตั้งค่าการแจกจ่ายรูปภาพสำหรับงานอีเวนต์
- **ระบบจดจำใบหน้า** - ช่วยให้ผู้เข้าร่วมค้นหารูปภาพของตัวเองได้อัตโนมัติ
