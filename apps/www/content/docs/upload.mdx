# คู่มือการอัปโหลด

SabaiPics รองรับหลายวิธีในการอัปโหลดรูปภาพงานอีเวนต์

## วิธีการอัปโหลด

### iOS WiFi Transfer

โอนรูปภาพโดยตรงจาก iPhone หรือ iPad ผ่าน WiFi

1. เปิดแอป SabaiPics iOS
2. เชื่อมต่อ WiFi เดียวกันกับคอมพิวเตอร์
3. เลือกรูปภาพที่ต้องการโอน
4. รูปภาพจะซิงค์โดยอัตโนมัติ

### อัปโหลดผ่านเบราว์เซอร์

อัปโหลดโดยตรงจากเว็บเบราว์เซอร์

1. ไปที่แดชบอร์ดงานอีเวนต์
2. คลิก "อัปโหลดรูปภาพ"
3. ลากและวางหรือเลือกไฟล์
4. รอให้ระบบประมวลผลเสร็จสิ้น

### แอปเดสก์ท็อป

ใช้แอปเดสก์ท็อปสำหรับอัปโหลดจำนวนมาก

### Lightroom Plugin

ส่งออกโดยตรงจาก Adobe Lightroom
